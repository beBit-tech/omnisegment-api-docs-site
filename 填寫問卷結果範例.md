# 填寫問卷結果範例

| **key** | **Description**       | **Sample** | **Required**    | **Data Type** | Note |
|:-------:| --------------------- | ---------- | --- | ------------- | ---- |
| subject | 該題目名稱            |            | &#10004; | string        |      |
|  type   | 該題目 type           |            | &#10004; | string        |      |
|   sn    | 對應到該問卷題目的 sn |            | &#10004; | int           |      |
| answer  | 該題目填寫的答案      |            | | list          | 注意項目排序裡，答案是有順序性的．在多選題跟重複核選題裡如果有 "其他" or "Other Option" 會去查找 otherAnswer 看答案． |
| otherAnswer  | 該題目填寫的"其他"答案      |            | | list          | 只有多選題跟重複核選題會需要該 key，當 answer 裡有 "其他" or "Other Option" 會查找該 key 看使用者填寫了什麼 |
