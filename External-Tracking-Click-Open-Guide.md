## Use Case
Whenever link/destination site url is not generated by Omnisegment, but by Third party service.

## How to inject tracking step by step

1. Third party API would be called with extra parameter "omnisegment_tracking_url", 
request payload should be like
```
{
"member_to_send": ...,
"edm_to_send": ...,
"authentication_stuff": ...,
"omnisegment_tracking_url": https://..
}
```
2. Combine target link/destination site url with omnisegment_tracking_url as paramter "goto", for instance:
```
omnsegment_tracking_url = "https://omnsegment.com/collect?v=1&t=event&dr=email&tid=oa-319523&ea=TrackingClick&ec=Tracking&luid=_135123125&tuid=1231412351_13251232_3412351"
destination_url = "https://shop/product/123"

combined_url = "https://omnsegment.com/collect?v=1&t=event&dr=email&tid=oa-319523&ea=TrackingClick&ec=Tracking&luid=_135123125&tuid=1231412351_13251232_3412351?goto=https%3A%2F%2Fshop%2Fproduct%2F123"

```
Remember to encode destination_url

3. shorten combined url and put it inside edm
